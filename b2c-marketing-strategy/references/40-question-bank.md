# 40-question priority bank

Authored 2026-04-27 as the basis for Mode 2 (7 questions) and Mode 3 (20 questions) selection. Ranked by **impact-per-question** = (number of dashboard sections the answer unblocks) × (depth of `★ hyp` → fact replacement). The 7 Mode 2 / 20 Mode 3 picks are made from this list — see `interview-questions.md` for the active sets.

If a future Mode 2/3 prod run shows a specific section persistently thin, this bank is the place to look for a question to promote.

---

## Tier A — foundational (1-10): each unblocks 1+ full section

1. **Customer segments** — 3-5 named cohorts: size, JTBD, demographics, frequency, sensitivity profile, geo. *Feeds:* #consumers, #audience-lifecycle, #comm-matrix, #media targeting, #regions. **Highest impact-per-question — closes half of brand-audience side of the doc in one answer.**
2. **Marketing envelope** — % of revenue OR absolute $ band. *Feeds:* #mmm, #media, exec objectives.
3. **Top 3 priorities for the year** — channels to grow, segments to acquire, problems to solve. *Feeds:* #exec, #product-initiatives, #comm-plan T1, #media reallocation. Steers narrative spine.
4. **Direct competitors top 4-5 + main risk per** in next year. *Feeds:* #competitors direct, #swot threats, #risks register, #positioning.
5. **Active brand tracker** — yes/no + latest wave (TOM, Aided, Consideration, Trial, NPS). *Feeds:* #brand-health entirely (otherwise under .hypothesis-badge).
6. **Last year's actual channel mix + ROAS / CAC by channel.** *Feeds:* #media channel-overview, #mmm ROAS column + saturation tags.
7. **Northstar + lagging KPIs + 2025 actuals + 2026 targets across funnel.** *Feeds:* #business-goal table, #exec objectives grid, #assumptions calibration.
8. **Top 3-5 flagship product launches** — name + 1-line + Q + budget + northstar metric. *Feeds:* #product-initiatives entirely + #comm-plan T1 anchoring.
9. **Brand pyramid foundations** — essence + purpose + 3-5 personality + 3-6 RTBs. *Feeds:* #brand-pyramid + #positioning emotional + #comm-matrix upper-funnel.
10. **Positioning statement** + functional axes — "We help X who Y to Z. Unlike A, we B." *Feeds:* #positioning hypothesis text + #exec asymmetries + #basics positioning.

## Tier B — scope-shapers (11-20): each narrower but meaningful

11. **Constraints — what's off-limits** creative / brand / category. *Feeds:* #media bucket exclusions, #comm-plan, creative briefs.
12. **Regional / city-tier strategy** — where defend, where expand. *Feeds:* #regions cluster table, comm-plan launch waves.
13. **2025 miss + lesson.** *Feeds:* #swot weaknesses, #assumptions trigger logic, #risks.
14. **Indirect competitors** — top 3 wallet-share / attention substitutes. *Feeds:* #competitors indirect, #risks substitution.
15. **Risk tolerance + top 3-5 specific risks ranked by P × I.** *Feeds:* #risks bubble chart, #assumptions trigger conditions.
16. **Lifecycle approach** — Awareness / Activation / Retention / Loyalty per cohort. *Feeds:* #audience-lifecycle 3×3 matrix.
17. **Big brand message / hook** per priority segment (≤40 chars). *Feeds:* #comm-matrix upper-funnel, creative briefs.
18. **Active vs lapsed cohort sizes** — addressable / MAU / lapsed 90+. *Feeds:* #audience-lifecycle row labels.
19. **Cultural / seasonal calendar** in market. *Feeds:* #comm-plan special projects, always-on cadence.
20. **Brand attribute scores vs top 2 competitors** (5-7 attributes). *Feeds:* #brand-health attribute table.

## Tier C — calibration (21-30): polish on existing sections

21. **Always-on streams + budget split** — performance / driver-recruitment / CRM cadence. *Feeds:* #comm-plan always-on, #mmm ongoing.
22. **Functional axes for positioning maps** in this category (2 dims that drive consumer choice). *Feeds:* #positioning functional plot.
23. **Emotional archetype target for 2026** — bold / caring / friendly / self-assured shift. *Feeds:* #positioning emotional plot 2026 target.
24. **Cohort retention curve** — M1 / M3 / M6 actuals. *Feeds:* #business-goal retention rows, #brand-health funnel.
25. **Take rate / commission / monetization mechanic.** *Feeds:* #competitors monetization row, #business-goal monetization stage.
26. **Partnerships in scope** — name + what we get + what they give. *Feeds:* #mmm partnerships bucket, flagship co-brand mentions.
27. **Measurement cadence** — weekly / monthly / quarterly review structure. *Feeds:* #risks monitoring, operating-review intro.
28. **LTV : CAC / contribution margin / payback period actuals.** *Feeds:* #business-goal profitability stage.
29. **Other roadmap features beyond flagships** (5-15 KANO classification). *Feeds:* #product-initiatives roadmap table.
30. **T1 vs T2 vs T3 campaign tier split** — how many of each planned. *Feeds:* #comm-plan grid, #media bucket allocation.

## Tier D — nice-to-have (31-40): specifics, niche, fillable from research

31. **Voice-of-customer quotes per segment** — from CS / NPS / qual sessions. *Feeds:* #consumers segment voice-quote.
32. **Cross-sell / upsell triggers internal** — when family-share / corporate / premium surface. *Feeds:* #audience-lifecycle existing-user message.
33. **App store rating distribution + review themes.** *Feeds:* #brand-health attribute calibration, #competitors app-rating row.
34. **Driver / supplier / two-sided cohort definitions.** *Feeds:* #consumers supply segment, #competitors supply-side overlap.
35. **Influencer partnerships current vs planned** — names of mid-tier creators on retainer. *Feeds:* #media influence bucket, #comm-plan T2.
36. **CRM performance metrics** — push CTR, email open, SMS opt-in. *Feeds:* #media CRM row, #mmm CRM bucket.
37. **Reactivation flow triggers + 30/60/90 day cadence.** *Feeds:* #audience-lifecycle churned-user channels.
38. **Macro assumptions** — FX / inflation / GDP / CAGR you're betting on. *Feeds:* #assumptions low-confidence row, #risks macro tile.
39. **Org-chart context** — DRIs for each strategic bet (role title, not personal name). *Feeds:* #risks owner column, #next-steps owner column.
40. **Past strategy doc / brand book / segmentation study** — final upload reminder if not already shared. *If file exists, this is upload-path, not interview question.*
