# Real estate marketplace

**Examples:** Cian, Zillow, Rightmove, Idealista, Zonaprop

**Typical segments:** Active buyers / renters, agents (small office / boutique / chain), developers, browsers (research-mode), landlords.

**Channel split benchmark:** Performance 35-45 / Brand 20-25 / Influence 5-10 / CRM 10-15 / Tools+Agencies 8-12 / Reserve 5

**KPI metrics:** Listings count, agent accounts, lead-to-agent rate, listing freshness, mortgage attach rate (if vertical).

**Category risks:** Macro housing cycle, regulator rental rules, agent commission compression, search-engine de-listing, FSBO (for-sale-by-owner) substitution.

**Strategically distinctive:** Highly local / cyclical. Brand affinity weak among agents (they multi-home). Mortgage / valuation cross-sells are the real margin.

**Radar axes (Competitors section):** Listing volume · Photo & 3D quality · Agent trust · Filter granularity · Price transparency · Map UX

**Brand-funnel benchmark (category leader, Mode 1 hypothesis anchor):** TOM 20–35% · Aided 60–80% · Consideration 35–50% · Trial / 90-day-active 20–35%
