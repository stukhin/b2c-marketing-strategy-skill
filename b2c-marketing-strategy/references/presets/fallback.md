# Out-of-list / first-principles fallback

If brand is outside all 15 supported + 6 adjacent categories — first-principles fallback.

**Approach:**

1. Use Tier 1 / Tier 2 / Tier 7 sources from `research-sources.md`.
2. Build a skeletal preset by analogy: which of the 15 most resembles this business in unit economics + supply structure?
3. Use that preset's channel split + KPI as starting hypothesis.
4. Mark all benchmarks ★.
5. Set user expectations explicitly: "hypothesis-heavy output, recommend follow-up validation with a brand tracker / category research".

**Example analogues:**
- Subscription mental-health app → Edtech preset structure (free-tier funnel, low completion rate, CRM-heavy)
- B2C SaaS productivity → Subscription streaming structure (paid subs, churn-driven, brand-as-content)
- Resale-marketplace for collectibles → Fashion / resale preset structure (community + cultural relevance > performance ROAS)
- Gig-economy adjacent (e.g., babysitting) → Two-sided marketplace + Ride-hail combined logic

If no analogue feels right, default to Classifieds marketplace structure (most generic two-sided pattern) + flag heavily.

**Radar axes & Brand-funnel benchmark:** copy from the analogue preset chosen above. If first-principles only, pick 6 axes that drive consumer choice in this category (price / quality / trust / convenience / discovery / mobile UX is a generic starting set) and use new-entrant benchmark anchors (TOM 5–15%, Aided 25–50%, Consideration 12–25%, Trial 5–15%). Mark all numbers `★ hyp`.
