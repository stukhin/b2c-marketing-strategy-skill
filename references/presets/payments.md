# Payments & remittance

**Examples:** Wise (B2C), PayPal, Remitly, WorldRemit, Western Union digital

**Typical segments:** Migrant remitters (corridor-specific), e-commerce buyers, freelancers receiving payouts, businesses paying suppliers (some B2C2B overlap).

**Channel split benchmark:** Performance 40-50 / Brand 15-20 / Influence 5-10 / CRM 10-15 / Tools+Agencies 8-12 / Reserve 5

**KPI metrics:** Transaction volume, average ticket, FX margin, repeat transaction rate, corridor coverage, settlement time.

**Category risks:** Regulator licensing per corridor, FX margin compression by stablecoins / CBDCs, incumbent partnership shifts (banks de-risking remitters), AML rule changes.

**Strategically distinctive:** Corridor-specific. Brand weight low except in trust-building creative. Performance ROAS very channel-dependent (immigrant TV dwarfs other in some corridors).

**Radar axes (Competitors section):** Fee level · FX rate / margin · Speed · Coverage (corridors) · Trust · Mobile UX

**Brand-funnel benchmark (corridor leader, Mode 1 hypothesis anchor):** TOM 10–22% · Aided 45–70% · Consideration 25–40% · Trial / first-transfer 15–30%
