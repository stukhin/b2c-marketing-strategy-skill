# Fashion / resale marketplace

**Examples:** Depop, Poshmark, ThredUp, Vestiaire Collective, The RealReal

**Typical segments:** Sellers (closet-cleaners / power-resellers / boutique flippers), buyers (deal-hunters / hardcore-resale / sustainability-driven / treasure-hunters), Gen Z baseline.

**Channel split benchmark:** Performance 30-40 / Brand 25-35 / Influence 15-20 / CRM 8-12 / Tools+Agencies 4-8 / Reserve 5

**KPI metrics:** GMV, active listings, repeat-buyer rate, authentication accuracy, take rate (often 0% to drive supply), cross-border share.

**Category risks:** Authentication scandals, shipping cost inflation, big-fashion-brand entry (incumbent acquisitions consolidating market), counterfeit floods.

**Strategically distinctive:** Highest brand+influence weight in B2C. Community / cultural relevance matters MORE than performance ROAS. Trust and authentication = product.

**Radar axes (Competitors section):** Selection breadth · Item authenticity · Pricing fairness · Buyer-seller trust · Returns / disputes · Style discovery

**Brand-funnel benchmark (category leader, Mode 1 hypothesis anchor):** TOM 15–25% · Aided 50–70% · Consideration 25–40% · Trial / 90-day-active 15–30%
