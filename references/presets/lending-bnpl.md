# Lending & BNPL

**Examples:** Klarna, Affirm, Afterpay, Tabby (MENA), Zip

**Typical segments:** Younger thin-file consumers (BNPL), credit-builders, prime cardholders, bank-credit-rejected (adjacent NBFI), merchants (B2B side).

**Channel split benchmark:** Performance 40-50 / Brand 15-25 / Influence 5-10 / CRM 12-18 / Tools+Agencies 5-8 / Reserve 3

**KPI metrics:** Approved applications, default rate, 90-day delinquency, GMV financed, merchant count, gross margin per user, regulatory capital usage.

**Category risks:** Regulator credit-reporting rules (BNPL increasingly under bank-grade rules), credit cycle / default surge, merchant churn, charge-off classification.

**Strategically distinctive:** Trust / responsibility narrative is non-optional. CRM-driven approval / re-engagement is the real moat. Default underwriting = product.

**Radar axes (Competitors section):** Approval speed · APR / fee transparency · Repayment flexibility · Merchant breadth · Trust · Mobile UX

**Brand-funnel benchmark (category leader, Mode 1 hypothesis anchor):** TOM 15–25% (Klarna-led developed markets) · Aided 50–75% · Consideration 30–45% · Trial / first-loan 20–35%
