# Travel / OTA

**Examples:** Booking, Expedia, Airbnb, Trip.com, MakeMyTrip

**Typical segments:** Leisure single / couple, family travelers, business travelers, last-minute, package vs unbundled, region-specific (domestic vs outbound).

**Channel split benchmark:** Performance 45-55 / Brand 15-20 / Influence 5-10 / CRM 8-12 / Tools+Agencies 8-12 / Reserve 3

**KPI metrics:** Bookings, total transaction value (TTV), repeat customers, take rate, attach rate (flight + hotel + insurance), cancellation rate.

**Category risks:** Geopolitical (visa, war, pandemic), supplier disintermediation (hotels direct), regulator transparency rules, AI travel agent disruption.

**Strategically distinctive:** Highest performance % in B2C — Google / Bing dominate funnel. Brand work crucial for repeat (vs comparison-shopping). Highly seasonal.

**Radar axes (Competitors section):** Inventory breadth · Price level · Booking UX · Cancellation / refund policy · Trust · Mobile UX

**Brand-funnel benchmark (category leader, Mode 1 hypothesis anchor):** TOM 25–45% (Booking-led) · Aided 70–90% · Consideration 40–55% · Trial / first-booking 25–40%
