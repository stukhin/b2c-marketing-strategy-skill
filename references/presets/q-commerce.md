# Quick-commerce / grocery

**Examples:** Getir, Gorillas (RIP), Flink, Instamart, GoPuff, Zapp

**Typical segments:** Urban professionals, parents (convenience-driven), late-night fillers, frequency-of-use cohorts.

**Channel split benchmark:** Performance 40-50 / Brand 20-25 / Influence 5-10 / CRM 12-18 / Tools+Agencies 5-8 / Reserve 3

**KPI metrics:** Orders / day / dark store, AOV, repeat rate, contribution margin per order, SKU count, fulfillment time.

**Category risks:** Unit economics challenge (sub-$5 contribution margin), full-shop substitute (weekly Tesco etc.), real estate cost for dark stores, labor regulation.

**Strategically distinctive:** Density-bound. Strategy = winning specific neighborhoods, not country-wide. CRM is core because category is high-frequency.

**Radar axes (Competitors section):** Selection breadth (SKUs) · Delivery time · Price level · Inventory accuracy · Trust · Mobile UX

**Brand-funnel benchmark (category leader in active city, Mode 1 hypothesis anchor):** TOM 12–25% · Aided 50–70% · Consideration 25–40% · Trial / 90-day-active 15–30%
